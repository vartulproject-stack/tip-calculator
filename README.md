## ✨ Features
- Calculate tip based on bill amount
- Custom tip percentage: 0%, 5%, 10%, 15%, 20%
- Split bill among multiple people
- Real-time calculation with validation
- Responsive design - Mobile friendly
- Clean gradient UI

## 🛠️ Tech Stack
- HTML5
- CSS3 - Flexbox, Gradient
- Vanilla JavaScript - DOM, Events, Functions

## 🎯 What I Learned
- DOM manipulation with getElementById
- Event handling with onclick
- Input validation for edge cases
- Mathematical operations in JS
- Responsive CSS design
- parseFloat & toFixed for decimals
